# Emacs Configuration

My personal Emacs configuration, built over 15 years. Targets Emacs 30+ with native compilation.

## Features

- **Package management**: Elpaca with use-package integration
- **Completion**: Vertico + Orderless + Marginalia + Embark + Consult + Corfu
- **Org-mode**: Agenda, org-roam, org-modern, citar, org-ref
- **Development**: LSP (Pyright), tree-sitter, Flycheck, DAP, Apheleia formatting
- **Git**: Magit with delta, diff-hl
- **Navigation**: Avy, Ace-window, Consult
- **Appearance**: Modus themes, doom-modeline, ligatures (Iosevka)

## Structure

- `early-init.el` — UI cleanup and startup performance
- `init.el` — Main configuration
